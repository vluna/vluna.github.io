AIDE

[ About the project ]

Our software is aiming to help end-user, which are people who are not professional software developer, develop for the web in an easier and more exploratory manner. This involves providing visuals that gives better context and structure for the end user by organizing their code in a block like shape in a stack structure. It is to help novice focus more on the logic of their program and less on how to use the editor. With the proposed digital interface, end-user will have the ability to explore different sets of solutions and can grasp a better context of the software with the metaphor, blocks of code.

FACE RECOGNITION

[ About the project ]

Our team created an application that allows the user to compare their face similarity with another person of their choice. We implemented an algorithm that takes particular coordinate positions and uses them to calculate facial proportions, such as the distance between eyes, width and height of the face, distance between eyebrows etc. It compares a multiple of distances, adds the values, scales them and returns the result in percentage. We used an already built face tracking library called clmtrackr and extended its features for face tracking. This is a javascript library for fitting facial models to faces in videos or images.

Vuoto

[ About the project ]

Vuoto, Survival in Space is a first person survival game in a space exploration setting, where there are different tasks according to the characters in the game. The genre we aimed for was an Action-Adventure and Strategy First Person Survival.

The game consists in that everyone's faith in the quest depends on their actions, your survival depends on your actions. It is a survival game that goes through objective based missions with a parallel storyline. However, the final destiny of the protagonist is in the hand of the player controlling it.

[ Teammates ]

  • Dhruv Arora
  • Kelvin Gonzales
  • Marina Khvan

[ Core objectives ]

  • Maintain life support
  • Maintain/increase oxygen support
  • Survive until ship gets fixed
  • Overcome challenges of terrain
  • Recover lost resources
  • Explore planet

[ Sub objectives ]

  • Gather as many items as he/she can find in order to explore the information about the entire crew members and being able to connect with them.
  • The main challenge for the player is to find the way to reach the end of the game without losing health or oxygen. If one of them reaches the critical level 0, the player dies. There are obstacles that the player has to overcome in each different level of the game. At the end of each level the player finds essential items - parts of the ship, which will help the crew to repair it and come back home.

[ Core loop ]

  • Explore Map: Players are able to explore the map. Find collectables. Read about crew members. Communicate with them.
  • Terrain Challenge: It will involve the player to interact with the environment like going through a maze and being precise with their first person controller. Save health until the last level.
  • Get Objective: Reach the destination on the map. Switch to the next level by grabbing the objective at the end of the map.

[ Smart Depth ]

  • Level: Terrain Challenges will differ in each level so it will be unique but using same mechanic and gameplay to solve or maneuver through it.
  • In addition to the existing level we added extra functionality which we hope will add to player’s sense of curiosity and connect the player to the narrative.
  • Gather Collectable Items: Items will be scattered around the map for player to search and gather. Through these items he will be connecting to the crew members through the diegetic UI elements on screen.
